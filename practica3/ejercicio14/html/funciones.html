Funciones en Python

Definición de Funciones

Las funciones en Python se definen utilizando la palabra clave def seguida del nombre de la función y los parámetros entre paréntesis. 


# Ejemplo de definición de función
def saludar(nombre):
    return f'Hola, {nombre}!'

Llamada a Funciones

Para llamar a una función y ejecutar su código, simplemente se utiliza el nombre de la función seguido de los paréntesis con los argumentos (si los hay). 


# Ejemplo de llamada a función
mensaje = saludar('Juan')
print(mensaje)  # Salida: Hola, Juan!

Parámetros y Argumentos

Las funciones pueden aceptar parámetros (valores que la función espera recibir) y estos pueden tener valores por defecto. 


# Ejemplo de función con parámetro opcional
def calcular_area(base, altura=1):
    return base * altura

area_rectangulo = calcular_area(5, 3)  # Se pasa altura como argumento
area_cuadrado = calcular_area(4)      # altura toma el valor por defecto

Retorno de Valores

Las funciones pueden retornar valores utilizando la palabra clave return. 


# Ejemplo de función que retorna múltiples valores
def calcular_area_perimetro(lado):
    area = lado ** 2
    perimetro = lado * 4
    return area, perimetro

area, perimetro = calcular_area_perimetro(5)

Funciones Recursivas

Las funciones en Python pueden llamarse a sí mismas, lo que se conoce como recursión. 


# Ejemplo de función recursiva para calcular factorial
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)
