Condicionales en Python

Operadores de Comparación

  • == : Igual a
  • != : No igual a
  • < : Menor que
  • > : Mayor que
  • <= : Menor o igual que
  • >= : Mayor o igual que

Operadores Lógicos

  • and : Retorna True si ambas condiciones son verdaderas.
  • or : Retorna True si al menos una de las condiciones es verdadera.
  • not : Retorna True si la condición es falsa.

Operadores de Membresía

  • in : Retorna True si encuentra un elemento específico en una secuencia.
  • not in : Retorna True si NO encuentra un elemento específico en una secuencia.

Operador de Identidad

  • is : Retorna True si ambos operandos apuntan al mismo objeto en memoria.
  • is not : Retorna True si ambos operandos NO apuntan al mismo objeto en memoria.

Expresiones Condicionales

Utiliza la forma expresion_si_verdadero if condicion else expresion_si_falso para asignar valores basados en una condición.

Comprobación de Vacío

  • None : Representa la ausencia de valor.
  • not objeto : Retorna True si el objeto es None, vacío, o falso.
