Listas en Python

Creación de Listas

Las listas en Python se crean utilizando corchetes [] y pueden contener cualquier tipo de datos. 


# Ejemplo de creación de una lista
mi_lista = [1, 2, 3, 'a', 'b', 'c']

Acceso a Elementos

Los elementos de una lista se acceden utilizando índices, comenzando desde 0. 


# Acceder al primer elemento de la lista
primer_elemento = mi_lista[0]

Operaciones Básicas

  • append() : Agrega un elemento al final de la lista.
  • extend() : Extiende la lista agregando elementos de otra lista.
  • insert() : Inserta un elemento en una posición específica.
  • remove() : Elimina el primer elemento de la lista que coincide con el valor especificado.
  • pop() : Elimina y retorna el elemento en la posición especificada (o el último si no se especifica).
  • index() : Retorna el índice del primer elemento con el valor especificado.
  • count() : Retorna el número de veces que aparece un elemento en la lista.
  • sort() : Ordena los elementos de la lista.
  • reverse() : Invierte el orden de los elementos de la lista.

Copiar Listas

Para copiar una lista en Python, se pueden utilizar métodos como copy() o rebanadas ([:]). 


# Ejemplo de copia de lista usando copy()
copia_lista = mi_lista.copy()

Listas y Bucles

Las listas son comúnmente utilizadas en bucles for para iterar sobre sus elementos. 


# Ejemplo de bucle for sobre una lista
for elemento in mi_lista:
    print(elemento)
